## 문제

예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.

## 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 출력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 예제 출력

```
*        *
**      **
***    ***
****  ****
**********
****  ****
***    ***
**      **
*        *
```

## 풀이

```
let fs = require("fs");
let input = fs.readFileSync("/dev/stdin").toString().trim().split('\n');

let n = Number(input[0]);

for(let i=1; i<=n; i++){
  //' '가 8 6 4 2 0
  console.log('*'.repeat(i) + ' '.repeat(2*n - 2*i) + '*'.repeat(i));
}
for(let i=n-1; i>=1; i--){
  //' '가 2 4 6 8
  console.log('*'.repeat(i) + ' '.repeat(2*n - 2*i) + '*'.repeat(i));
}
```
